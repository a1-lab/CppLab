1) создали пустой проект
2) заходим "свойства проекта"/компоновщик/система
   устанавливаем параметр "Подсистема" в Windows.
3) Добавляем класс Application
4) Параметры проекта/Свойства конфигурации/Общие
   "Стандарт языка С++" С++17
5) Параметры проекта/Свойства конфигурации/"С/С++"
   "Дополнительные каталоги включаемых файлов"
   добавляем а)$(WXWIN)\include;
             б)$(WXWIN)\include\msvc 
   WXWIN - переменная среды. Указывает на католог со 
           скомпилированными wxWidgets.
6) Параметры проекта/Свойства конфигурации/Компоновщик
   "Дополнительные каталоги библиотек"
